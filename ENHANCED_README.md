# bsc-yield-aggregator-vaults

Auto-compounding yield farming vaults for major BSC protocols.

## Overview
Starter for building yield aggregators and auto-compounders on BNB Chain (BSC).

## Tech Stack
- Solidity
- Hardhat
- PancakeSwap / other BSC DEXes

## Getting Started
```bash
git clone https://github.com/Proxtox/bsc-yield-aggregator-vaults.git
cd bsc-yield-aggregator-vaults
npm install
```

## Key Files
- `contracts/YieldVault.sol` - Basic share-based vault with deposit/withdraw
